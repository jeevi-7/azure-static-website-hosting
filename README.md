# static Website Hosting on Azure Storage
## Live Website URL
https://newsamplee.z13.web.core.windows.net/
## Project Overview
This project demonstrates how to deploy a static website using Azure Storage Account’s Static Website Hosting feature.
Azure automatically gives a public endpoint where the website becomes available to anyone.
## Architecture
*Azure Storage Account
*Blob Service (Static Website)
*$web container
*Public Endpoint
## Steps Implemented
*Created a Storage Account in Azure
*Enabled Static Website Hosting
*Uploaded website files (index.html, error.html)
*Configured public access for Blob service
*Deployed website to Azure’s global endpoint
*Tested deployment
